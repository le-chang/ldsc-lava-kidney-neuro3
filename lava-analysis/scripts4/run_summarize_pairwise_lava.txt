## What This Script Does:

1. **Reads all pairwise results** (egfr_pd, hematuria_pd, uacr_pd)
2. **Generates comprehensive summaries**:
   - Total tests and significant findings per pair
   - Top significant results with genomic locations
   - Chromosome distribution
   - Effect directions (positive/negative correlations)
3. **Creates comparison with LDSC** global results
4. **Saves multiple output files**:
   - Combined tables for all analyses
   - Filtered significant results only
   - Detailed text report
   - Summary statistics table
